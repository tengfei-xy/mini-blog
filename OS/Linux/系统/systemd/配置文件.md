配置文件：/etc/systemd/user.conf、/etc/systemd/system.conf

```ini
# 最大文件打开数量
DefaultLimitNOFILE=1048576
# 核心转储文件最大字节数
DefaultLimitCORE=infinity
```

修改完成后

```
systemctl daemon-reexec
```

此时查看资源的进程量

```
cat /proc/PID/limits
```
