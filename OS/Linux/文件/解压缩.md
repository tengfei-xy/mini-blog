# zip

输出文件名sql.zip

```
zip -r sql.zip ./*
```

解压覆盖

```
unzip -o test.zip
```

压缩到指定目录

```
unzip -o -d /temp test.zip
```

解压指定文件

```
unzip *.zip "a.txt"
```

# rar

压缩：将所有.jpg的文件压缩成一个rar包，并处于名为all.rar，该程序会将.rar 扩展名将自动附加到包名后。

```
rar a all *.jpg
```

解压：e

```
unrar x file.rar
```

# zst

```
yum install zstd

```

## 7z

安装7z

```
yum install -y p7zip
```

解压

```bash
# 直接解压
7za x hashcat-6.2.5.7z
# 指定目录
7za x hashcat-6.2.5.7z -o ~
```
