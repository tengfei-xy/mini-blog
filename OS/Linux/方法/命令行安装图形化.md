## 安装

```
yum groupinstall "Server with GUI" -y
```

## 直接启动

默认情况下以root权限启动

```
startx
```

## 开机设置

```
systemctl get-default                   # 查看当前启动模式
systemctl set-default graphical.target  # 启动模式更改为图形化界面模式
systemctl set-default mulit-user.target # 动模式更改为命令行界面模式
```

## 设置其他用户也能启动GUI

```bash
# touch  /var/run/console/<用户名>
```
