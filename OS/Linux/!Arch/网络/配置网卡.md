# 接口

查看网络接口

```
ip link
```

启动网络接口

```
ip link set ens33 up
```

# IP

设置静态IP

 ```
 ip address add 192.168.119.200/24 dev ens33
 ```

# 路由

查看ipv4的路由

```
ip route show
```

查看ipv4的路由

```
ip -6 show
```

添加路由

```
ip route add 192.168.119.0/24 ens33
```

添加默认路由

```
ip route add default via 192.168.119.2 dev ens33
```



删除路由

```
ip route del 192.168.119.0/24 ens33
```
