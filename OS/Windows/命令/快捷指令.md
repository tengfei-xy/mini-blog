# 快捷指令

------

以下快捷命令从Win+R中进行使用

# 控制面板的cpl 系列

| 名称           | 快捷打开    |
| -------------- | ----------- |
| 系统属性       | sysdm.cpl   |
| 网络连接       | ncpa.cpl    |
| 鼠标属性       | main.cpl    |
| 防火墙基本配置 | firewal.cpl |

# 常用

| 名称             | 快捷打开               |
| ---------------- | ---------------------- |
| 控制面板         | control                |
| 计算机基本属性   | control system         |
| 用户账户         | control userpasswords2 |
| 资源监视器       | resmon                 |
| 查看系统激活转态 | xlmgr.vbs /xpr         |
| 查看系统信息     | cmd /k systeminfo      |
| 注册表           | regedit                |
| 文件资源管理器   | explorer               |
| 磁盘清理         | cleanmgr               |
| 系统信息         | msinfo32               |
| 任务管理器       | taskmgr                |
