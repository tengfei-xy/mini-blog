下载版本

```
wget https://download.redis.io/releases/redis-6.0.9.tar.gz
```

解压

```bash
tar zxf redis-6.0.9.tar.gz && cd redis-6.0.9
```

安装依赖(执行2次)

```
sudo yum -y install centos-release-scl devtoolset-9-gcc devtoolset-9-gcc-c++ devtoolset-9-binutils scl-utils gcc automake autoconf libtool make
```

scl方式进入版本的高版本的gcc环境

```bash
# 如果gcc小于5.3版本，临时有效，退出shell会恢复原 gcc 版本
scl enable devtoolset-9 bash

# 其他方式
# https://zhuanlan.zhihu.com/p/382390995
# sudo yum install devtoolset-9-gcc-c++
# source /opt/rh/devtoolset-9/enable
```

开始编译

```
sudo make PREFIX=/usr/local/redis-6.2.6 install
```

> 编译错误：
>
> 如遇到Error jemalloc/jemalloc.h: No such file or directory when making Redis
>
> 输入 make distclean

进入程序目录，并创建目录

```
cd /usr/local/redis-6.2.6
sudo mkdir conf data logs
```

直接启动不会在前台输出日志，需要查看日志文件

```
cd /usr/local/redis-6.2.6
nohup bin/redis-server conf/redis6378.conf &
```

最后建议加入supervisor，参考redis.ini

```
[program: redis6379]
command=taskset -c 1 /usr/local/redis-6.2.6/bin/redis-server /usr/local/redis-6.2.6/conf/redis6379.conf
process_name=%(program_name)s
numprocs=1
directory=/usr/local/redis-6.2.6
umask=022
priority=999
autostart=true
startsecs=5
startretries=3
autorestart=true
exitcodes=0,2
stopsignal=kill
stopwaitsecs=10
stopasgroup=false
killasgroup=false
user=nbzhiwei
redirect_stderr=true
stdout_logfile=/usr/local/logs/redis/redis-6379-info.log
stdout_logfile_maxbytes=50MB
stdout_logfile_backups=10
stdout_capture_maxbytes=1MB
stdout_events_enabled=false
stderr_logfile=/usr/local/logs/redis/redis-6379-error.log
stderr_logfile_maxbytes=10MB
stderr_logfile_backups=10
stderr_capture_maxbytes=1MB
stderr_events_enabled=false
```
