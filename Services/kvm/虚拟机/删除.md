关机

```
virsh destroy vm
```

查找虚拟机的映像文件，并删除

```
virsh dumpxml win10-1 | grep "/.*qcow2" -o
```

删除定义

```
virsh undefine vm
```



删除脚本

```shell
#!/bin/bash
name="$1"
test -z "${name}" && { echo "未指定虚拟机名" ; exit 1}
sudo virsh destroy "${name}"
{ sudo virsh dumpxml "${name}" | grep "/.*qcow2" -o | xargs -n1 sudo rm -rf ;}  && sudo virsh undefine "${name}"
exit 0
```
