| 指令         | 说明                   | 举例                |
| ------------ | ---------------------- | ------------------- |
| b address    | 在某物理地址上设置断点 | b 0x7c00            |
| c            | 继续执行，直到断点     | c                   |
| s            | 单步执行               | s                   |
| info cpu     | 查看寄存器信息         | info                |
| r            | 查看寄存器信息         | r                   |
| sreg         | 查看寄存器信息         | sreg                |
| creg         | 查看寄存器信息         | creg                |
| xp /nuf addr | 查看物理地址内容       | xp /10bx 0x100000   |
| x /nuf addr  | 查看线性地址内容       | x /40wd 0x9000      |
| u start end  | 反汇编一端内存         | u 0x100000 0x100010 |
