## stat的状态字符

| 字母 | 解释                                            |
| -- | --------------------------------------------- |
| D  | 无法中断的休眠状态（通常 IO 的进程）                          |
| R  | 正在运行可中在队列中可过行的                                |
| S  | 处于休眠状态                                        |
| T  | 停止或被追踪                                        |
| W  | 进入内存交换 （从内核2.6开始无效）                           |
| X  | 死掉的进程 （基本很少见）                                 |
| Z  | 僵尸进程                                          |
| <  | 优先级高的进程                                       |
| N  | 优先级较低的进程                                      |
| L  | 有些页被锁进内存；                                     |
| s  | 进程的领导者（在它之下有子进程）； 优先级较低的进程                    |
| l  | 多线程，克隆线程（使用 CLONE\_THREAD, 类似 NPTL pthreads）； |
| +  | 位于前台的进程组；                                     |
