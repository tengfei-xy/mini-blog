# grep

| posix       | 说明                  |
| ----------- | ------------------- |
| \[:alnum:]  | 文字数字                |
| \[:alpha:]  | 字符                  |
| \[:digit:]  | 数字                  |
| \[:graph:]  | 非空字符（非空格、控制字符）      |
| \[:lower:]  | 小写字符                |
| \[:upper:]  | 大写字符                |
| \[:cntrl:]  | 控制                  |
| \[:print:]  | 可打印字符（包含空格)         |
| \[:punct:]  | 标点符号                |
| \[:space:]  | 所有空白字符（新行，空格，制表符）   |
| \[:xdigit:] | 十六进制数字（0-9，a-f,A-F） |
