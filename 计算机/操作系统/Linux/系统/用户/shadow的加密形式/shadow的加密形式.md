# shadow的加密形式

对于/etc/shadow文件

| 开头的字段             | 加密形式     |
| ----------------- | -------- |
| ****$1$、****$md5$ | MD5      |
| ​$2$              | blowfish |
| ​$5$              | SHA-256  |
| ​$6$              | SHA-512  |
|                   |          |

shadow文件密码存储格式：$id$salt$encrypted$

-   的开头的字段为加密
-   的开头的字段为
-   \*为的加密
