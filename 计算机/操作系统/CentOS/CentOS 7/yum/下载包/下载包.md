## 下载已安装的依赖

-   repotrack方式
    1.  安装yum-utils
        ```纯文本
        yum install yum-utils
        ```
    2.  下载包(下载到当前工作路径)
        ```纯文本
        repotrack <包名>
        ```
-   yum方式
    ```bash
    yum reinstall --downloadonly --downloaddir=<路径> <包名>
    ```
-   yumdownloader方式
    1.  下载yumdownloader
        ```bash
        yum install yum-utils
        ```
    2.  下载包
        ```纯文本
        yumdownloader --resolve --destdir <下载路径> <包名>
        ```
