# Ctrl系列快捷键

## 移动光标

| 快捷键          | 功能说明       |
| ------------ | ---------- |
| ctrl+A       | 光标移动到行首    |
| ctrl+E       | 光标移动到行尾    |
| ctrl+Insert  | 复制命令行      |
| Shift+Insert | 粘贴命令行      |
| ctrl+k       | 剪切光标到行尾    |
| ctrl+u       | 剪切光标到行首    |
| ctrl+w       | 剪切光标前的一个单词 |
| ctrl+y       | 粘贴剪切或删除的文本 |
