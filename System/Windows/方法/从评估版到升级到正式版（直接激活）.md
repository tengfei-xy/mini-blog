# 从评估版到升级到正式版（直接激活）

获取系统版本方法

```、
PS> DISM /online /Get-CurrentEdition
部署映像服务和管理工具
版本: 6.2.9200.16384

映像版本: 6.2.9200.16384

当前版本为:

当前版本 : ServerDatacenterEval

操作成功完成。
```

注：如果当前版本有Eval结尾的，需要去掉

激活命令

```
PS> DISM /online /Set-Edition:ServerDatacenter  /ProductKey:<KEY> /AcceptEula
```
